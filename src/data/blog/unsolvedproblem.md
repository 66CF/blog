# 解决不了的问题

“先生，这是您第153次询问这个单词的意思了。”

“哦，鲍勃，请你再告诉我一次吧。”

这是来自2048年的数百米高的大厦顶端的办公室里的对话。显然，佐二汀先生又在阅读《史记》了。事实上，他非常喜欢在工作之余读一读《史记》。但很可惜，他看不懂文言文，甚至一窍不通。好在，他的机器人“鲍勃”可以解决这些问题。

“先生，这个‘之’被放在主谓之间，用于取消句子的独立性。需要我深入解析吗？”

佐二汀先生微微一笑，拒绝了它的好意。他说：“那么现在，这个问题就成功解决了。”

这时，有人敲门。在佐二汀先生让鲍勃喊了“请进”之后，那人才恭恭敬敬地走进来。原来他是佐二汀先生的手下，是来汇报一些重要工作的。

“帮我记下来，鲍勃。哦，这是上面的新文件。天呐，它应该放在哪个柜子来着……好吧，鲍勃，你应该记得，帮我放进去吧。真搞不懂这些人为什么还执着于纸质文件，麻烦……”

在人工智能的“帮助”下，佐二汀处理完了工作。他正想再读一读《史记》，就听到系统提示音：“先生，您和忧尔茂先生先生预定的见面时间到了，地点是您的办公室。”

佐二汀先生一听是在自己的办公室会面，非常高兴。不过他是什么时候约的忧尔茂先生的呢？于是他问鲍勃。鲍勃则很耐心地告诉他，他们是五年的老朋友，而今天是他们相见的五周年纪念日，所以他们昨天约好了见面。

昨天吗？佐二汀先生表示自己有点印象。于是他吩咐鲍勃弄些他们都喜欢吃的食物，顺便打扫一下卫生。

“您今早刚吩咐过我打扫卫生。先生，请问需要再打扫一次吗？”

佐二汀先生环顾四周，也不知道房间到底算不算干净整洁，只好说：“你若是觉得还可以，就算了。”

于是鲍勃分析了一遍房间整洁度，决定不再打扫。

很快，忧尔茂先生来了。佐二汀先生赶紧让鲍勃替自己表达高兴之情。忧尔茂先生也让他的人工智能“麦克”说了类似的话。他们看着两个机器人替自己拥抱着对方，感到十分满足。

忧尔茂先生吃了顿饭便离开了。佐二汀先生闲下来，便继续读他的《史记》。鲍勃则如已往替他翻译。这是他第14遍读《史记》了，他的后台记录着。

“嘿，鲍勃，这个‘之’是什么意思？”

“先生，这是第154次了。它用于主谓之间……”

“好吧。这次，这个问题真的解决了……”

*问题解决了吗？大概暂时是的，然而人类所面临的问题仍在增加，那就是过度依赖。*
